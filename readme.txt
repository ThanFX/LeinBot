Лотерейный бот.

Бот представляет собой обычный исполняемый файл LeiNBot.exe. После запуска открывается консоль вывода служебной информации. Для закрытия бота достаточно закрыть эту консоль. В консоль могут публиковаться отдельные критичные ошибки.
Для успешного запуска необходимо, чтобы рядом с исполняемым файлом находился файл настроек - prop.json. Параметры:
token - особый идентификатор дискордного бота. Берётся на странице регистрации приложений Дискорда https://discordapp.com/developers/applications/ после создания и регистрации бота.
guilds, channels - идентификаторы гильдий и их каналов, сообщения из которых бот будет отслеживать. Идентификаторы просто получить, открыв Дискорд в браузере и взяв URL страницы. Так, например, URL канала "#чат" гильдии LeiN имеет следующий вид: https://discordapp.com/channels/535838700769378335/535838701251592214, первой число - это идентификатор гильдии, второе - идентификатор канала. Идентификаторов гильдий и каналов может быть любое количество.
admins - список пользователей Дискорда, на команды которых бот будет реагировать. Используется полный вариант дискордного ника, состоящего из непосредственно никнейма, разделителя и цифрового тега. Можно узнать в профиле пользователя.
При изменении настроек бота необходимо перезапустить, чтобы изменения вступили в силу.
Важно!! Без этого файла или если он повреждён, запуск бота не произойдёт!!

Принцип работы
Для начала создаётся список пользователей, участвующих в розыгрыше - имена или короткие ники пользователей Дискорда. Список сохраняется в формате csv (каждое имя на отдельной строчке), в кодировке UTF-8 without BOM. Название файла строго предопределено: lottery-N.csv, где N это номер проводимой лотереи. Одновременно могут существовать произвольное количество списков розыгрышей. Все файлы можно редактировать и во время работающего бота, отдельно перезапускать его не нужно.

Всё взаимодействие с ботом происходит путём отправки специальных сообщений в любой канал, который разрешён боту в настройках. Такие сообщения должны начинаться с ключевого слова "!lottery", затем команда и параметры через пробел. Так, например, все текущие доступные команды можно посмотреть командой "!lottery help". Бот отвечает в тот канал, в котором была отправлена команда. Таким образом можно сделать все настройки и проверки розыгрышей в одном канале, закрытом от участников, а потом провести только розыгрыш специальной командой уже в другом канале.

У большинства команд ключевым параметром, задаваемым после самой команды, является номер розыгрыша, цифра, которая соответствует номеру в названии файла со списком участников.

Список команд, их параметров и описание:
!lottery list
Отобразит список всех файлов со списками участников

!lottery check N - (N - номер розыгрыша)
Проверит розыгрыш с заданным номером:
- Ищёт всех участников розыгрыша следи членов гильдии и выдаёт уведомление, если такие не найдены
- Сравнивает количество участников розыгрыша и количество призовых мест в розыгрыше и выдаёт уведомление, если призовых мест больше (все гарантированно получат по призу)

!lottery params N type tour|count|prize
Задаёт параметры розыгрыша с заданным номером. Прочие параметры:
type - тип розыгрыша (турнир с местами, лототрон с билетами, еtc). Сейчас реализован только первый тип, поэтому всегда должно быть "tournament".
tour - номер призового места (3-2-1 и т.д.)
count - количество победителей на этом месте
prize - приз победителю, текстовая строка
Например, сообщение "!lottery params 1 tournament 3|5|50к золота" задаст для розыгрыша правила "На третьем месте розыгрыша будет 5 победителей, каждый из которых получит по 50к золота".
Количество мест и победителей в них может быть любым, но места должны следовать по порядку, до первого. Т.е. может быть 5-4-3-2-1 места, но не могут быть 5-2-1 места.

!lottery status N
Выводит информацию о текущих параметрах розыгрыша с заданным номером (тип, места, количество победителей, призы для каждого места).

!lottery start N
Запускает розыгрыш с заданным номером. Случано определяются победители в каждом из мест и выводится уведомление.
Важно! Команду для каждого розыгрыша нужно запускать только 1 раз, при повторном запуске произойдёт новый расчёт победителей и вывод уведомления!!

!lottery help
Выводит краткую справку по командам бота.
